# Frontend Web App

Next.js frontend application for Synect.

## Tech stack

- Next.js 16 (App Router)
- React 19
- TypeScript

## Run

```bash
bun run dev
```

App runs at `http://localhost:3000` by default.

## Build and start

```bash
bun run build
bun run start
```

## Lint

```bash
bun run lint
```
